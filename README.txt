Monte_carlo.py=======================================
This is a financial health-check tool that uses different approaches to find a user's financial situation, given various user-supplied inputs. 
Currently the tool employs iterative and monte-carlo simulation based approaches.
Compatible with Python 3.6


Profile_classifier.R=======================================
This is a demo financial profile classifier written using R.
It uses logistic regression to classify a user's income level compared to demographic statistics in Taiwanese Dollars
Compatible with R 3.5.0
